# Fonts Collection.

我們統整了一些常用且實用的字體們，
希望能分享給所有的人。

下面我們依照不同字體的特色進行分類。

- Serif Fonts: 襯線字體。
- Sans Serif: 無襯線字體。
- Mono Space: 等寬字體。
- Hand Write: 手寫字體。

## Serif 襯線字體。

> 襯線，經典也超越經典。

襯線字體富含文學氣息，

讓使用者在享受字體的美之餘，也能清晰辨識內容。

## Sans Serif 無襯線字體。

> 無襯線，現代且美觀。

無襯線字體，符合現代設計美學。

簡潔、易識是他的特點。

## Mono Space 等寬字體。

> 特殊設計，符合特殊的你。

等寬字體，不限制於襯線 or 不襯線。

特殊的字體設計符合不同使用場景，讓你的設計抓住觀賞者的眼球。

## Hand Write 手寫字體。

> 原汁原味，追求極致的美感。

手寫字體，最大化的給予使用者筆觸的力於美。

手寫的溫暖，讓現今社會不只在紙上能感受的到。
